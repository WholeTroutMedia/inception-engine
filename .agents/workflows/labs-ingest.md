# /labs-ingest Workflow

## Overview

This workflow governs the ingestion of exploratory artifacts generated by Google Labs tools (Mixboard, Flow, NotebookLM, Genie) into the Creative Liberation Engine. Since these tools lack automated API extraction paths, this workflow structures a human-in-the-loop ingestion point that immediately fires autonomous follow-up tasks.

## The Ingestion Vectors

1. **Visual Assets (Mixboard, Flow, Imagen)**
    * **Source:** Concept boards, UI sketches, cinematic renders.
    * **Target Directory:** `_assets/labs/visual/`
    * **Next Action:** Fed into `DESIGN` mode. PRISM will use Gemini's multimodal capabilities to analyze the visual structure and implement the design system constraints (ATELIER).

2. **Structural RAG (NotebookLM)**
    * **Source:** Audio Overviews, synthesized Markdown from massive document sets.
    * **Target Directory:** `_assets/labs/structural/`
    * **Next Action:** Fed into `PLAN` mode or `SCRIBE`. The synthesized logic is converted into actionable `HANDOFF.md` specifications.

3. **Spatial Logic (Project Genie)**
    * **Source:** 2D platformer logic, voxel concepts.
    * **Target Directory:** `_assets/labs/spatial/`
    * **Next Action:** Fed into `comet-nexus` or AIA scripting for Rooms.xyz.

## Execution Steps

If the user invokes this workflow, immediately execute the following:

1. **Acknowledge and Orient:**
    Acknowledge the intent to ingest Labs artifacts.
2. **Verify Staging:**
    Check `_assets/labs/` for new files (images, markdown, json).
3. **Determine Vector:**
    Based on the file type and user prompt, identify the correct ingestion vector (Visual, Structural, Spatial).
4. **Trigger Automation:**
    * If **Visual**: Auto-start `/design` or `/design-edit` citing the new asset as the primary inspiration.
    * If **Structural**: Auto-start `/plan` or invoke `SCRIBE` to index the knowledge.
    * If **Spatial**: Auto-start world-building scripts.

> **Zero-Day Principle:** The goal is to move from a shiny visual artifact on a phone screen to a deployed, styled web route with zero intermediate friction.
