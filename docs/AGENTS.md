## Light Edition vs Full Brainchild V4

This repo ships **15 core agents** — the essential team that covers every phase of the IDEATE → PLAN → SHIP → VALIDATE workflow. The full Brainchild V4 system runs 35 agents with strict validation gates, production infrastructure, and 5 isolated validator agents.

| | Light Edition (this repo) | Full Brainchild V4 |
|---|---|---|
| **Total agents** | 15 | 35 |
| **Hives** | 4 | 6 |
| **Validators** | Checklist-based | 5 isolated validator agents |
| **Source of truth** | inception-engine | brainchild-v4 (private) |

Everything in the Light Edition is designed to receive updates pushed down from V4 over time.

## How to Call an Agent

Just use their name with `@`. Works in any LLM or IDE AI assistant:

```
@AVERI boot — starts a session with full strategic awareness
@ATHENA — architecture question
@BOLT — build this component
@COMET — open this URL and extract the data
@KEEPER — what do we know about this topic?
@LEX — is this approach legally sound?
@COMPASS — does this pass constitutional review?
```

You don't need to manually route agents. `@AVERI` will coordinate the right specialists automatically based on what you're asking.
