## How to Call an Agent

Just use their name with `@`. Works in any LLM or IDE AI assistant:

```
@AVERI boot — starts a session with full strategic awareness
@ATHENA — architecture question
@BOLT — build this component
@COMET — open this URL and extract the data
@KEEPER — what do we know about this topic?
@LEX — is this approach legally sound?
@COMPASS — does this pass constitutional review?
```

You don't need to manually route agents. `@AVERI` will coordinate the right specialists automatically based on what you're asking.
