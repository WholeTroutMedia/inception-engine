# RunwayML Suite

RunwayML is the most comprehensive cloud-based generative video editor. It has evolved from a single model provider to a multi-model generative hub.

## Account Status

- **Status:** Logged In
- **Credits:** 1,685 available

## Internal Editing Suite: Generative Session

The Runway app groups work into "Sessions" providing a full NLE-style approach to generation.

- **Multi-Model Support:** Access Gen-4.5 alongside third-party models like Kling 3.0 directly in the dashboard.
- **Reference Management:** Sidebar specifically for managing image references to guarantee character and visual consistency across multiple shots.
- **Tools:** Motion Sketch (drawing paths for movement), Character Render, and Lip Sync.

## Usage in Creative Liberation Engine

The core external workhorse for the `ffmpeg-swarm`. When the user requests a "Transmission" or complex video sequence, Runway is the primary API/Web endpoint used to generate the raw footage before local post-processing.
