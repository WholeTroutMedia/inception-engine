# LottieFiles Design Library

LottieFiles is the industry standard for lightweight, scalable vector animations on the web. It uses the Lottie format (exported from After Effects or created via their platform) to render complex UI motion with tiny file sizes.

## Account Status
- **Status:** Logged In (Profile Active)

## Key Features
- **UI Micro-interactions:** Buttons, loaders, success states.
- **Motion Tokens:** Bind live data to text, colors, and transforms inside the animation.
- **Performance:** Much lighter than video or GIF counterparts.
- **React Ready:** Ships with `lottie-react` components.

## Usage in Creative Liberation Engine
LottieFiles serves as the web counterpart to DaVinci/Foley Engine. While the Media Pipeline handles heavy video/audio, Lottie handles frontend UI motion at 60fps independent of resolution.
