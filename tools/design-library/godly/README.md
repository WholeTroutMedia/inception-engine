# Godly Design Reference

Godly is a highly curated gallery of the world's best interactive websites, focusing on premium, motion-heavy, and high-fidelity design.

## Details
- **Curated By:** Daryl Ginn
- **Database:** 1,000+ premium sites
- **Aesthetic:** Tends toward dark mode, webgl, scroll-jacked storytelling, and ultra-sleek SaaS.

## Usage in Creative Liberation Engine
Godly is used strictly as a reference point for **Creative Quality**. When producing designs via IDEATE or DESIGN modes, Godly serves as the high-water mark for aesthetic judgment. The bar for Creative Liberation Engine output should mirror the curation standards of this platform.
