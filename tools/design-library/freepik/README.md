# Freepik Design Hub

Freepik has evolved from a pure stock asset library into a comprehensive generative design suite.

## Account Status

- **Status:** Logged In (Premium)

## Internal Editing Suite: Freepik Designer (Beta)

Freepik Designer provides a web-based canvas (similar to Canva or simplified Figma) tailored for AI asset manipulation.

- **AI Presentation:** Generate full slide decks.
- **Manipulation:** Built-in background removal and multi-model image generation.
- **Integration:** Drag-and-drop millions of vectors, PSDs, and photos directly into the generative workspace.

## Usage in Creative Liberation Engine

Used by IRIS when generating static pitch decks, mockups, or composite images where stock vectors need to be blended with generative AI imagery before hitting the `design-tokens` layer.
