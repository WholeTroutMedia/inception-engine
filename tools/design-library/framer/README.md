# Framer Platform

Framer is a professional web builder that outputs performant React applications. Unlike pure component libraries (like Relume), Framer is an end-to-end canvas environment.

## Usage in Creative Liberation Engine
While the Creative Liberation Engine builds React apps from raw code, Framer serves two purposes:
1. **Component Pipeline:** Sourcing complex motion components via the Framify layer.
2. **Prototyping:** Extremely rapid, high-fidelity prototypes that behave exactly like production code because they *are* production code under the hood.
