# Spline Design Library

Spline is a web-based 3D design tool designed for creating interactive 3D experiences that drop directly into React applications. It bridges the gap between `spatial-core` and web interfaces.

## Account Status
- **Workspace:** jaharroni's Workspace
- **Plan:** Free Tier

## Key Features
- **React Integration:** Export ready-to-use `<Spline>` components for Next.js/React.
- **AI 3D Generation:** Generate geometry from text prompts.
- **Interactivity:** State machines for hover, click, and scroll-driven 3D animations.
- **Physics Engine:** Real-time physics for web components.

## Usage in Creative Liberation Engine
Use Spline for heroic landing page elements, interactive 3D objects instead of flat 2D illustrations, and spatial computing bridges. Export the URL and use `@splinetool/react-spline`.
