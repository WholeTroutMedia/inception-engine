# Kling AI Studio

Kling AI provides advanced text-to-video and image-to-video generation capabilities.

## Account Status

- **Status:** Logged In
- **Plan:** Pro Account

## Internal Editing Suite: Kling Canvas

Unlike basic prompt boxes, Kling utilizes the **Kling Canvas**.

- **Role:** Acts as an "Intelligent Creation Agent".
- **Capabilities:** Users can describe complex ideas (e.g., "generate a 1-minute cyberpunk storyboard"), and the canvas manages the layout.
- **Specific Tools:** Multi-Angle Storyboard, Product Visual Generator, and standard sequential storyboarding.

## Usage in Creative Liberation Engine

Kling AI is the primary external node for high-fidelity, temporally consistent video generation within the Media Pipeline, particularly when multi-angle references are required for a product or character.
