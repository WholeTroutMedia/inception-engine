## First Boot

```bash
git clone https://github.com/Creative-Liberation-Engine/creative-liberation-engine.git
cd creative-liberation-engine
pnpm install
cp .env.example .env   # Add GEMINI_API_KEY (or GOOGLE_API_KEY)
pnpm run build
pnpm run dev           # Boots on http://localhost:4100
```

In a second terminal:

```bash
pnpm run genkit:ui     # Opens http://localhost:4000 — explore the flows
```
