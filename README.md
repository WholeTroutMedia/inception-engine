# cle

a generative reasoning engine.

---

processes start. processes run. processes end.
some of them make things.

---

## processes

| process | function |
|---------|----------|
| `kstratd` | strategic reasoning layer |
| `kverifyd` | validation and truth-checking layer |
| `kexecd` | execution and build layer |
| `kmemorypd` | persistent memory layer |
| `kwrited` | write and archive layer |
| `kauditd` | constitutional compliance layer |
| `kshadowd` | parallel QA layer |

---

## start

```bash
npm install
npm run dev
```
